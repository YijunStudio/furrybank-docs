User 用户

必填：nickname、email、password。

新增用户时，MySQL 触发器将在 Collection 表和 Account 表中新增 id 一致的相应记录，且目标记录的 user_id 字段亦为新增的用户 id。

属性 MySQL类型 名称 允许空 默认值 备注
nickname VARCHAR(100) ×
email VARCHAR(100) ×
phone VARCHAR(36)
gender TINYINT(1) ×
password VARCHAR(100) ×
status TINYINT(1) ×
locale CHAR(36) ×
